2. List Interface

ArrayList vs LinkedList

FeatureArrayListLinkedList
Internal StructureDynamic ArrayDoubly Linked List
Random AccessO(1) ✅O(n) ❌
Insert/Delete at middleO(n)O(1) if iterator
MemoryLess overheadMore (prev+next refs)
// ArrayList Example
ArrayList<String> list = new ArrayList<>();
list.add("Java");
list.add("Python");
list.get(0);        // "Java"
list.remove(1);     // removes "Python"
list.size();        // 1

3. Map Interface

// HashMap Example
HashMap<String, Integer> map = new HashMap<>();
map.put("Alice", 90);
map.put("Bob", 85);
map.get("Alice");         // 90
map.containsKey("Bob");   // true
map.keySet();             // [Alice, Bob]

// Iterating
for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}
💡 Tip: Use HashMap for O(1) lookups, TreeMap for sorted keys, LinkedHashMap for insertion-order.

4. Common Algorithms

Collections.sort(list);           // Sort a list
Collections.reverse(list);        // Reverse
Collections.shuffle(list);        // Random order
Collections.max(list);            // Find max
Collections.frequency(list, "x"); // Count occurrences
