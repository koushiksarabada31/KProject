Subject: CS202 | Topic: Combinational Logic Circuits

Slide 1: Introduction to Combinational Circuits

A combinational circuit's output depends only on the current inputs (no memory/state).

Examples: Adders, Subtractors, Multiplexers, Decoders, Encoders

Slide 2: Half Adder

ABSum (S)Carry (C)
0000
0110
1010
1101

S = A ⊕ B (XOR)    C = A · B (AND)

Slide 3: Full Adder

Adds three bits: A, B, and Carry-in (Cin)

Sum = A ⊕ B ⊕ Cin

Cout = (A · B) + (Cin · (A ⊕ B))

Slide 4: Multiplexer (MUX)

A 2ⁿ-to-1 MUX selects one of 2ⁿ inputs using n select lines.

4:1 MUX: Y = S̄₁·S̄₀·I₀ + S̄₁·S₀·I₁ + S₁·S̄₀·I₂ + S₁·S₀·I₃

Slide 5: Decoder

An n-to-2ⁿ decoder activates exactly one of 2ⁿ outputs.

2:4 Decoder outputs: D₀ = S̄₁·S̄₀, D₁ = S̄₁·S₀, D₂ = S₁·S̄₀, D₃ = S₁·S₀

Slide 6: Karnaugh Map (K-Map)

A graphical method to simplify Boolean expressions.

B̄C̄B̄CBCBC̄
Ā1011
A1001

Steps: 1) Fill K-Map → 2) Group adjacent 1s (powers of 2) → 3) Write simplified expression
