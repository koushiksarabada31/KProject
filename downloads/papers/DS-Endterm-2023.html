Part A — Short Answer (5 × 2 = 10 Marks)

[2M]1. Define Abstract Data Type (ADT). Give one example.
Solution: An ADT is a mathematical model for data types where the data type is defined by its behavior (operations) rather than its implementation. Example: Stack ADT with push(), pop(), peek() operations.
[2M]2. What is the difference between stack and queue?
Solution: Stack uses LIFO (Last In First Out) — push/pop at same end. Queue uses FIFO (First In First Out) — enqueue at rear, dequeue at front.
[2M]3. Write the time complexity of binary search.
Solution: Best: O(1), Average: O(log n), Worst: O(log n). Space: O(1) iterative, O(log n) recursive.
[2M]4. What is a complete binary tree?
Solution: A binary tree where all levels are fully filled except possibly the last level, which is filled from left to right.
[2M]5. State the property of a max-heap.
Solution: In a max-heap, the value of each node is greater than or equal to the values of its children. The root contains the maximum element.

Part B — Long Answer (4 × 10 = 40 Marks)

[10M]6. Implement a singly linked list in Java/C with insert, delete, and display operations.
Solution: 
class Node {
    int data; Node next;
    Node(int d) { data = d; next = null; }
}
class LinkedList {
    Node head;
    void insertAtHead(int data) {
        Node newNode = new Node(data);
        newNode.next = head;
        head = newNode;
    }
    void deleteKey(int key) {
        if (head != null && head.data == key) { head = head.next; return; }
        Node prev = head;
        while (prev.next != null && prev.next.data != key) prev = prev.next;
        if (prev.next != null) prev.next = prev.next.next;
    }
    void display() {
        Node curr = head;
        while (curr != null) { System.out.print(curr.data + " -> "); curr = curr.next; }
        System.out.println("null");
    }
}
[10M]7. Explain the working of Quick Sort with an example. Analyze its time complexity.
Solution: Quick Sort uses divide-and-conquer with a pivot element. Steps: 1) Choose pivot 2) Partition: elements < pivot go left, > pivot go right 3) Recursively sort left and right. Time: Best/Avg O(n log n), Worst O(n²) when already sorted. Space: O(log n).
[10M]8. Construct a BST from: 50, 30, 70, 20, 40, 60, 80. Show inorder, preorder, and postorder traversals.
Solution: Inorder (sorted): 20, 30, 40, 50, 60, 70, 80 | Preorder: 50, 30, 20, 40, 70, 60, 80 | Postorder: 20, 40, 30, 60, 80, 70, 50
[10M]9. Compare BFS and DFS graph traversals. Write pseudocode for BFS.
Solution: BFS uses queue (level-order), DFS uses stack/recursion (depth-first). BFS finds shortest path in unweighted graphs. 
BFS(graph, start):
    queue = [start]
    visited = {start}
    while queue is not empty:
        v = queue.dequeue()
        process(v)
        for each neighbor u of v:
            if u not in visited:
                visited.add(u)
                queue.enqueue(u)

Part C — Problem Solving (2 × 10 = 20 Marks)

[10M]10. Evaluate the postfix expression: 6 2 3 + * 8 2 / -
Solution: Step by step: Push 6, Push 2, Push 3 → Pop 2,3: 2+3=5, Push 5 → Pop 6,5: 6*5=30, Push 30 → Push 8, Push 2 → Pop 8,2: 8/2=4, Push 4 → Pop 30,4: 30-4=26. Answer: 26
[10M]11. Apply Dijkstra's algorithm to find shortest path from vertex A in the given graph.
Solution: Initialize dist[A]=0, all others=∞. Process minimum distance vertex, update neighbors. Continue until all vertices processed. Dijkstra's gives O((V+E)log V) with min-heap.
