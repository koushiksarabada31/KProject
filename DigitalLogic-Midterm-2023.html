Subject: Digital Logic Design (CS202) | Time: 1.5 Hours | Max Marks: 30

Part A — Short Answer (5 × 2 = 10 Marks)

[2M]1. Convert (156)₁₀ to binary.
Solution: 156 ÷ 2 repeatedly: (10011100)₂
[2M]2. State De Morgan's Theorem.
Solution: (A·B)' = A' + B' and (A+B)' = A'·B'
[2M]3. What is a don't care condition in K-Map?
Solution: A don't care (X) is an input combination that will never occur or whose output doesn't matter. Can be treated as 0 or 1 to simplify the expression.
[2M]4. Write the truth table for XOR gate.
ABA⊕B
000
011
101
110
[2M]5. What is the difference between combinational and sequential circuits?
Solution: Combinational: output depends only on current inputs (no memory). Sequential: output depends on current inputs AND past state (has memory elements like flip-flops).

Part B — Long Answer (2 × 10 = 20 Marks)

[10M]6. Simplify F(A,B,C,D) = Σm(0,1,2,5,8,9,10) using K-Map.
Solution: After grouping in 4-variable K-Map: F = B'D' + A'C'D' + A'B'D. Groups: (0,1,8,9)=B'D', (0,2,8,10)=needs checking, final simplified: F = B'D' + A'C'
[10M]7. Design a 4-bit binary adder using full adders. Explain carry propagation.
Solution: Connect 4 full adders in cascade. FA0: A0,B0,Cin=0→S0,C0. FA1: A1,B1,C0→S1,C1. FA2: A2,B2,C1→S2,C2. FA3: A3,B3,C2→S3,Cout. Carry ripples from LSB to MSB, causing propagation delay.
