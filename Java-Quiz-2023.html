Quiz Examination — September 2023

Subject: OOP with Java (CS203) | Time: 30 Minutes | Max Marks: 20

[2M]1. What are the 4 pillars of OOP?
A: Encapsulation, Inheritance, Polymorphism, Abstraction
[2M]2. Difference between abstract class and interface in Java?
A: Abstract class can have constructors, instance variables, and concrete methods. Interface (pre-Java 8) only has abstract methods and constants. A class can implement multiple interfaces but extend only one abstract class.
[2M]3. What is method overloading vs overriding?
A: Overloading: same name, different parameters (compile-time polymorphism). Overriding: same signature in subclass (runtime polymorphism).
[2M]4. Explain the 'final' keyword in Java.
A: final variable = constant, final method = can't override, final class = can't extend.
[2M]5. What is the difference between ArrayList and LinkedList?
A: ArrayList: dynamic array, O(1) random access, O(n) insert/delete. LinkedList: doubly-linked nodes, O(n) access, O(1) insert/delete at known position.
[2M]6. What is exception handling? Name two checked exceptions.
A: Mechanism to handle runtime errors using try-catch-finally. Checked: IOException, SQLException.
[2M]7. What is the difference between == and .equals() in Java?
A: == compares references (memory addresses). .equals() compares content/values (can be overridden).
[2M]8. Explain 'this' keyword with an example.
A: 'this' refers to the current object instance. Used to: access instance variables when shadowed by parameters, call other constructors, pass current object as argument.
[2M]9. What is JVM? What is its role?
A: Java Virtual Machine. Executes Java bytecode, provides platform independence ("Write Once, Run Anywhere"), manages memory via garbage collection.
[2M]10. What are access modifiers in Java?
A: public (everywhere), protected (package + subclass), default/package-private (same package only), private (same class only).
