Lecture 1: Eigenvalues and Eigenvectors

Definition

For a square matrix A, if Av = λv for some non-zero vector v, then:

  • λ is an eigenvalue of A
  • v is the corresponding eigenvector

Finding Eigenvalues

Solve: det(A - λI) = 0 (characteristic equation)

Example: A = [4  1]
             [2  3]

det(A - λI) = (4-λ)(3-λ) - 2 = 0
λ² - 7λ + 10 = 0
λ₁ = 5, λ₂ = 2

Lecture 2: Diagonalization

A matrix A is diagonalizable if: A = PDP⁻¹

Where P = matrix of eigenvectors, D = diagonal matrix of eigenvalues.

Key Property: Aⁿ = PDⁿP⁻¹ — this makes computing matrix powers very efficient!

Lecture 3: Cayley-Hamilton Theorem

Every square matrix satisfies its own characteristic equation.

If the characteristic equation is λ² - 7λ + 10 = 0, then A² - 7A + 10I = 0.
